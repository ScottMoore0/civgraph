
# ni_votes

End-to-end pipeline for Northern Ireland election transfer simulation and
referendum projection. The referendum stack now ships with a structured
data ingestion layer, temporal feature engineering, a multi-output ridge
model, per-party breakdown reporting, and full web/CLI integration
tests.

- See `config.py` to set I/O paths and model settings.
- Refer to `docs/referendum_simulator.md` for detailed workflow,
  automated test, and manual validation guidance.
